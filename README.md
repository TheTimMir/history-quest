# History Quest | Game part
---
## Setup
```
npm install
npm install -g firebase-tools
```

Login into Firebase:
```
firebase login
```

### Building
W/ automatic deployment to Firebase:
(must be logged in)

```
npm run deploy:all
```
---
Or just build the project:
```
npm run build
```
<br>

### Developement
To run in dev:
```
npm run dev
```
